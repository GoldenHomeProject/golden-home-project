## Case 1 — Andreass / newfitnessgadgets.com (2014, Wealthy Affiliate blog)

| Field | Value |
|---|---|
| Niche | Fitness gadgets, sub-focus swimming equipment |
| Site age at first sale | ~weeks (started 2014-09-01, first sale within weeks of launch) |
| First-sale traffic source | Reddit (r/Gadgets + r/Swimming), Friday-evening drop |
| Volume | ~13,000 weekend visitors, 422 Amazon clicks, **6 sales** |
| Click-through-to-Amazon | 422 / 13,000 = **3.2%** |
| Click-to-sale | 6 / 422 = **1.4%** |
| Product | Finis Neptune underwater MP3 player (single product post) |
| Tactic | Posted *review link*, not direct affiliate link. Curiosity-driven headline. Read subreddit rules first. |

**GHP comparison:** GHP has 521 Amazon clicks / 0 orders → 0% click-to-sale vs Andreass's 1.4%. If GHP had hit Andreass's rate, 521 clicks → 7 sales. The gap is not "we need more clicks." The gap is **the clicks we're sending are wrong**: wrong intent, wrong landing offer, or wrong audience.

Source: [wealthyaffiliate.com case](https://my.wealthyaffiliate.com/andreass/blog/first-amazon-sales-thanks-to-reddit-reddit-is-gold)

## Cross-case patterns (what actually worked)

1. **Intent-mode keywords matter more than authority.** Every case that earned within weeks targeted buy-mode searches: "best X under $Y", "Finis Neptune review", brand-vs-brand. None earned from informational top-of-funnel content first. **GHP has done this correctly** — the COVER cluster targets buy-mode intent. So this isn't the gap.

2. **First traffic was either Reddit or Pinterest, not Google.** Sandbox-aware operators didn't wait for SEO. They went where humans already were. GHP has done IG/YT/FB but NOT Reddit or Pinterest (per current channel mix). **This is a gap.** Reddit specifically converted at 1.4% for Andreass; GHP at 0% across IG/YT.

3. **Single-product deep-review pages outperformed multi-product roundups for first sales.** Andreass posted one product review on Reddit. The Female Marketer shared a blog post (singular). GHP has 8 COVER posts, but the spread may dilute the share-link advantage of having one undeniable hero piece.

4. **Most first-attempt sites fail.** Regpaq made nothing on sites 1 and 2; revenue came from site 3. GHP being on its first cluster is the *expected* shape. Stage-1 pivot to Cluster 1 (pet mattress protectors) on 2026-06-05 if 0 impressions is consistent with the base rate.

5. **The 3-qualifying-sale FB-group tactic exists.** Not honorable, but real. Useful if GHP's Amazon Associates account is nearing 180-day deactivation.

## What does NOT match the cases (and what that tells us)

- **None of the cases had a content engine producing daily reels for IG/YT.** GHP's biggest infrastructure investment (the flywheel) has zero precedent in this sample. The cases all converted via *single human-shared posts* in niche communities, not content automation. This corroborates the 2026-04-30 flywheel-as-progress documented failure mode — automated volume without audience does not reproduce what manual single-shot posting in target communities does.
- **None of the cases used multi-platform parallel attribution.** They picked one channel and shipped. GHP's parallel-tracks mode (2026-05-11) increases optionality but each individual channel is getting less focus than the cases that succeeded with one channel.

- **Will not unilaterally post to Reddit subreddits.** This is a new outreach surface that requires the user's explicit go-ahead. Drafting a Reddit-test plan into the queue is appropriate; executing it without authorization is not.
